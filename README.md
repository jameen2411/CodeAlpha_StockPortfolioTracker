# 📊 Stock Portfolio Tracker (Python)

A simple command-line Python project that allows users to simulate stock investments, track a portfolio, and export transaction data to a CSV file.

---

## 🧠 Features

- Buy simulated stocks (AAPL, MSFT, TSLA)
- Calculates investment value in real time
- Tracks total portfolio investment
- Stores all transactions in a CSV file
- Handles multiple purchases of the same stock
- Simple command-line interface
